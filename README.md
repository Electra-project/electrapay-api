## Contribute

### Getting Started

```bash
go get github.com/ruannelloyd/electrapay-api
cd GOPATH/ruannelloyd/electrapay-api
make install
```

Then copy/paste the `.env.sample` to `.env` and update the values.

### Run locally

```bash
make start
```

### Test

```bash
make test
```
